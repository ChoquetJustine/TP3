IOS

Histoire

Pour coder une application iOs, il faut :

  • Maîtriser Xcode
    Xcode est l'environnement de programmation pour toutes les applications iOs
  • Avoir un ordinateur iOs (iMac)
    Il n'existe aucune solution pour utiliser Xcode sous Windows ou Linux.
  • Posséder un bon éditeur de texte type Brackets ou SublimeText ainsi qu'un programme de création d'image vectorielle type Illustrator
  • Maîtriser le langage de programmation Objective -C ou celui qui là suivit, Swift, qui est plus moderne et a une syntaxe plus simple
  • Avoir un compte de développeur sur iOs Dev Center (payant : 99$ par an)
  • Penser son design, ses fonctionnalités, son ergonomie... pour une réussite de l'application
